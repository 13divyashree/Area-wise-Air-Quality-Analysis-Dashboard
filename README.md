# 🌍 Area-wise Air Quality Analysis Dashboard

## Description
A data visualization dashboard analyzing air quality across different areas to identify pollution levels and trends.

## Tools Used
- Power BI
- Data Visualization
- Data Analysis

## Key Insights
- Certain cities have higher AQI indicating severe pollution
- Clear difference between most and least polluted areas
- AQI varies across regions and over time
